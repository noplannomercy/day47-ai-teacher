@SRS.md

## 프로젝트 요약
- 디지털 교과서 포탈에서 AI Tutor PoC 데모 앱
- 고1 수학 개인화 튜터링 시스템의 프론트엔드
- 백엔드(n8n + Mem0 + pgvector + Wolfram)는 이미 완성됨
- 프론트엔드만 만들면 됨

## 핵심 포인트
1. 페이지 2개: 랜딩(/) + 채팅(/chat)
2. Supabase Auth 이메일/비밀번호 로그인 (데모 계정 4개 사전 생성됨)
3. 채팅은 n8n webhook POST 호출 → 응답 표시
4. AI 응답에 LaTeX 수식 포함 → KaTeX 렌더링 필수
5. 로그인 시 Mem0 API에서 학생 약점 데이터 조회 → 좌측 사이드바 표시
6. 로그인 시 n8n_chat_histories 테이블에서 이전 대화 로드
7. Vercel 배포

## 기술 스택 (확정)
- Next.js 15 + TypeScript + App Router
- shadcn/ui + Tailwind CSS
- Supabase Auth (@supabase/supabase-js, @supabase/ssr)
- KaTeX (react-katex)
- Vercel 배포
