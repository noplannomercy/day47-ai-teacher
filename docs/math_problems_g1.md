### [문제 1] 이차방정식의 근 구하기 (인수분해) [QE-001] (난이도: 하)

이차방정식 x² - 5x + 6 = 0 의 두 근을 구하시오.

[태그] 이차방정식, 인수분해, 근구하기, 하

**풀이:**
- 좌변을 인수분해하면: (x - 2)(x - 3) = 0
- x - 2 = 0 또는 x - 3 = 0
- 따라서 x = 2 또는 x = 3

**정답: x = 2 또는 x = 3**

---

### [문제 2] 이차방정식의 근 구하기 (완전제곱식) [QE-002] (난이도: 하)

이차방정식 x² - 6x + 9 = 0 의 근을 구하시오.

[태그] 이차방정식, 완전제곱식, 중근, 하

**풀이:**
- 좌변을 인수분해하면: (x - 3)² = 0
- x - 3 = 0
- 따라서 x = 3 (중근)

**정답: x = 3 (중근)**

---

### [문제 3] 근의 공식을 이용한 이차방정식 풀기 [QE-003] (난이도: 중)

이차방정식 2x² + 5x - 3 = 0 의 두 근을 구하시오.

[태그] 이차방정식, 근의공식, 판별식, 중

**풀이:**
- 근의 공식: x = (-b ± √(b² - 4ac)) / 2a
- a = 2, b = 5, c = -3
- 판별식 D = 25 - 4(2)(-3) = 25 + 24 = 49
- x = (-5 ± √49) / 4 = (-5 ± 7) / 4
- x = (-5 + 7)/4 = 1/2 또는 x = (-5 - 7)/4 = -3

**정답: x = 1/2 또는 x = -3**

---

### [문제 4] 판별식을 이용한 근의 개수 판단 [QE-004] (난이도: 중)

이차방정식 x² - 4x + 5 = 0 의 근의 개수를 판별하시오.

[태그] 이차방정식, 판별식, 근의개수, 허근, 중

**풀이:**
- 판별식 D = b² - 4ac
- D = (-4)² - 4(1)(5) = 16 - 20 = -4
- D < 0 이므로 실근이 없다.
- 따라서 서로 다른 두 허근을 가진다.

**정답: 실근 없음 (D = -4 < 0, 두 허근)**

---

### [문제 5] 근과 계수의 관계 [QE-005] (난이도: 중)

이차방정식 x² - 7x + 12 = 0 의 두 근의 합과 곱을 구하시오.

[태그] 이차방정식, 근과계수의관계, 비에타공식, 중

**풀이:**
- 근과 계수의 관계에 의해:
- 두 근의 합 = -b/a = 7
- 두 근의 곱 = c/a = 12
- 검증: (x-3)(x-4) = 0, 근은 3과 4, 합=7, 곱=12

**정답: 두 근의 합: 7, 두 근의 곱: 12**

---

### [문제 6] 근과 계수의 관계 활용 [QE-006] (난이도: 상)

이차방정식 x² - 5x + 3 = 0 의 두 근을 α, β라 할 때, α² + β² 의 값을 구하시오.

[태그] 이차방정식, 근과계수의관계, 대칭식, 상

**풀이:**
- 근과 계수의 관계: α + β = 5, αβ = 3
- α² + β² = (α + β)² - 2αβ
- = 25 - 6 = 19

**정답: α² + β² = 19**

### [문제 7] 이차방정식의 근의 조건 [QE-007] (난이도: 상)

이차방정식 x² + (k-1)x + k + 2 = 0 이 중근을 가질 때, 상수 k의 값을 구하시오.

[태그] 이차방정식, 판별식, 중근조건, 미정계수, 상

**풀이:**
- 중근 조건: 판별식 D = 0
- D = (k-1)² - 4(k+2) = 0
- k² - 2k + 1 - 4k - 8 = 0
- k² - 6k - 7 = 0
- (k-7)(k+1) = 0
- k = 7 또는 k = -1

**정답: k = 7 또는 k = -1**

---

### [문제 8] c가 음수인 이차방정식 [QE-008] (난이도: 중)

이차방정식 3x² - 2x - 5 = 0 의 두 근을 구하시오.

[태그] 이차방정식, 근의공식, c음수, 부호주의, 중

**풀이:**
- 근의 공식: a=3, b=-2, c=-5
- D = (-2)² - 4(3)(-5) = 4 + 60 = 64
- 주의: c가 음수이므로 -4ac = -4(3)(-5) = +60
- x = (2 ± 8) / 6
- x = 5/3 또는 x = -1

**정답: x = 5/3 또는 x = -1**

### [문제 9] 이차방정식 만들기 [QE-009] (난이도: 상)

두 근이 2 + √3, 2 - √3 인 이차방정식을 구하시오. (x²의 계수가 1)

[태그] 이차방정식, 근과계수의관계, 무리수근, 상

**풀이:**
- 두 근의 합 = 4
- 두 근의 곱 = (2+√3)(2-√3) = 4-3 = 1
- x² - 4x + 1 = 0

**정답: x² - 4x + 1 = 0**

### [문제 10] 이차방정식의 실생활 활용 [QE-010] (난이도: 상)

직사각형의 가로가 세로보다 3cm 길고, 넓이가 70cm²일 때, 세로를 구하시오.

[태그] 이차방정식, 실생활활용, 넓이, 상

**풀이:**
- 세로=x, 가로=x+3
- x(x+3) = 70
